## 这套资料适合怎么学

这套资料的定位不是“快速上手教程”，而是偏架构剖析。它更适合拿来建立下面这套心智模型：

- Agent Harness(智能体运行时骨架) 到底由哪些模块组成。
- Claude Code 这一类终端 Agent(智能体) 为什么不是“聊天机器人加几个工具”。
- 对话循环、工具系统、权限管线、记忆系统、上下文管理、Hook(钩子)、MCP(Model Context Protocol，模型上下文协议) 是怎么串起来的。
- 如果以后自己做 Agent，哪些模块能先简化，哪些模块不能省。

- 一张总架构图

- 一份“我自己的最小 Agent 设计草图”
